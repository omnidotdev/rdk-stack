# Reality Development Kit (RDK)

A unified API for web-based extended reality and immersive experiences.

## Services

| Service | Description |
|---------|-------------|
| [rdk](https://github.com/omnidotdev/rdk) | Core library |
| [rdk-site](https://github.com/omnidotdev/rdk-site) | Website |

## Getting Started

1. Copy configuration template:

   ```sh
   cp services.yaml.template services.yaml
   ```

2. Start development:

   ```sh
   tilt up
   ```
